## brawlstarsapi

BrawlStars & BrawlApi.com API wrapper, written on TypeScript

### Installation
#### NPM
```bash
npm i brawlstarsapi
```

#### PNPM
```bash
pnpm add brawlstarsapi
```

#### BUN
```bash
bun add brawlstarsapi
```
