## brawlapi

BrawlApi.com API wrapper, written on TypeScript

### Installation
#### NPM
```bash
npm i brawlapi
```

#### PNPM
```bash
pnpm add brawlapi
```

#### BUN
```bash
bun add brawlapi
```
