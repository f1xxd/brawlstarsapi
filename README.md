## brawlapi.com

BrawlApi.com API wrapper, written on TypeScript

### Installation
#### NPM
```bash
npm i brawlapi.com
```

#### PNPM
```bash
pnpm add brawlapi.com
```

#### BUN
```bash
bun add brawlapi.com
```
